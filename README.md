# MiniRedis

A Redis-inspired in-memory key-value database built in modern C++.

## Features (Planned)

- In-memory key-value store
- Command parser
- TCP server
- Multiple client support
- Thread-safe operations
- Key expiration (TTL)
- Persistence
- HTTP API

## Project Structure

- `include/` - Header files
- `src/` - Source files
- `tests/` - Unit tests
- `docs/` - Documentation

## Build

```bash
mkdir build
cd build
cmake ..
cmake --build .
```
